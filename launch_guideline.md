## Launch Guide
To launch the app: \
You should have node.js installed (version 19+)

## Frontend Launch
Run `npm run dev` in `./otps-test/`

## Backend Launch
Google Sheet Version: Run `node ./GoogleSheetsAPI.cjs` in `./otps-test/src/`
Firebase Version: Run `node ./FirebaseAPI.cjs` in `./otps-test/src/`
